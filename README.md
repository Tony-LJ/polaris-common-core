## 常用命令
```.text
创建项目
poetry new polaris_common_core
poetry初始化，会在项目根目录生成pyproject.toml文件
poetry init
添加与更新依赖项
poetry add requests
poetry update
只想更新特定的依赖项
poetry update requests
想安装 requests 的最新版本，而不受当前版本约束的限制
poetry add requests@latest
删除依赖
poetry remove requests
查看已安装的依赖
poetry show
查看环境信息
poetry env info
```
